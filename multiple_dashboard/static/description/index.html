This module allows you to have multiple dashboards that you can share with other users.
You can also create a menu to see a specific dashboard and put it wherever you want.

Dashboard container

Create dashboard container as much as you want

- You will find these options in: Dashboards > My Dashboards

When adding a new dashboard you can choose the container in which you want to put the new dashboard.

Share dashboard

You can share your dashboard by adding the user in your dashboard container.

Create a menu dashboard

Allow user to create a dashboard menu.

- You can create a dashboard menu from the Dashboard container.
